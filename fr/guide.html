Demarrage rapide

Bienvenue sur Vibe ! Voici comment commencer a utiliser l'application en quelques etapes simples.

  1. Telechargez l'application
    Vibe est disponible sur l'App Store (iOS) et Google Play Store (Android).
  2. Creez votre compte
    Inscrivez-vous avec votre email ou connectez-vous via Google ou Apple pour un acces rapide.
  3. Completez votre profil
    Ajoutez vos photos, votre nationalite et vos interets de voyage pour que les autres voyageurs puissent vous connaitre.
  4. Explorez la carte
    Decouvrez les voyageurs presents ou a venir dans votre destination.
  5. Connectez-vous
    Envoyez des demandes de contact et commencez a echanger !

Creer votre profil

Un profil complet augmente vos chances de connecter avec d'autres voyageurs.

Informations essentielles

Photo de profil

Choisissez une photo claire ou vous etes reconnaissable. Les photos de groupe ou les images sans visage reduisent la confiance des autres utilisateurs.

Prenom

Utilisez votre vrai prenom pour faciliter les echanges authentiques avec d'autres voyageurs.

Nationalite

Indiquez votre pays d'origine. Cela aide les autres voyageurs a vous trouver et a engager la conversation.

Date de naissance

Votre age sera calcule et affiche aux autres utilisateurs. Seules les personnes de 18 ans et plus peuvent utiliser l'application.

Personnaliser votre profil

  • Biographie : Decrivez-vous en quelques phrases. Parlez de vos passions, de vos experiences de voyage preferees.
  • Objectifs de voyage : Selectionnez ce que vous recherchez (compagnons de voyage, conseils locaux, activites, etc.)
  • Interets : Ajoutez vos centres d'interet pour trouver des voyageurs similaires.
  • Photos supplementaires : Ajoutez jusqu'a 6 photos pour montrer votre personnalite.
Conseil : Un profil complet avec plusieurs photos et une biographie detaillee recoit jusqu'a 5 fois plus de demandes de contact !

La carte interactive

La carte est le coeur de Vibe. Elle vous permet de visualiser les voyageurs du monde entier.

Types de pins sur la carte

Pin vert

"Ici maintenant" - Le voyageur est actuellement present a cet endroit.

Pin bleu

"Bientot la" - Le voyageur prevoit de se rendre a cet endroit prochainement.

Interagir avec les pins

  1. Appuyez sur un pin pour voir un apercu du profil
  2. Appuyez sur "Voir le profil" pour acceder au profil complet
  3. Depuis le profil, vous pouvez envoyer une demande de contact

Filtrer la carte

Utilisez les filtres pour affiner votre recherche :

  • Par ville ou destination
  • Par dates de voyage
  • Par type de presence (ici maintenant / bientot la)

Creer un voyage

Partagez vos projets de voyage pour que d'autres voyageurs puissent vous trouver.

Etapes de creation

  1. Choisissez votre destination
    Selectionnez la ville ou vous vous rendez. Vous pouvez indiquer si la destination est definitive ou encore en reflexion.
  2. Definissez les dates
    Indiquez vos dates d'arrivee et de depart prevues.
  3. Ajoutez des details (optionnel)
    Budget estime, activites prevues, ce que vous recherchez.
  4. Publiez votre voyage
    Votre voyage sera visible sur la carte et dans la section Discover.

Gerer vos voyages

Depuis la section "Mes voyages", vous pouvez :

  • Voir tous vos voyages actifs et passes
  • Modifier les details d'un voyage
  • Supprimer un voyage
  • Voir qui s'interesse a votre voyage

Messagerie

La messagerie vous permet de communiquer avec les voyageurs avec qui vous etes connecte.

Demarrer une conversation

  1. Visitez le profil d'un voyageur
  2. Appuyez sur "Envoyer un message" ou "Vibe"
  3. Si l'utilisateur accepte, vous pourrez echanger des messages

Fonctionnalites de la messagerie

  • Messages texte : Envoyez des messages instantanes
  • Indicateur de lecture : Voyez quand vos messages sont lus
  • Notifications : Recevez des alertes pour les nouveaux messages
Rappel de securite : Ne partagez jamais d'informations financieres ou de donnees sensibles via la messagerie.

Flash Meetups

Les Flash Meetups sont des rencontres spontanees organisees par les voyageurs.

Creer un Flash Meetup

  1. Definissez le lieu
    Choisissez un lieu public pour la rencontre (cafe, place, monument, etc.)
  2. Choisissez l'heure
    Les Flash Meetups sont pour des rencontres dans les prochaines heures.
  3. Ajoutez une description
    Decrivez l'activite ou le but de la rencontre.
  4. Definissez le nombre de participants
    Limitez si necessaire le nombre de personnes.

Rejoindre un Flash Meetup

Les Flash Meetups disponibles apparaissent sur la carte et dans la section Discover. Appuyez sur un meetup pour voir les details et rejoindre.

Conseil : Les Flash Meetups sont ideaux pour rencontrer des voyageurs rapidement quand vous etes deja sur place !

Securite et blocage

Votre securite est notre priorite. Voici les outils a votre disposition.

Bloquer un utilisateur

Si un utilisateur vous importune :

  1. Accedez a son profil ou a la conversation
  2. Appuyez sur le menu (trois points)
  3. Selectionnez "Bloquer"
  4. Confirmez le blocage

L'utilisateur bloque ne pourra plus vous contacter ni voir votre profil. Cette action est definitive.

Signaler un utilisateur

Pour signaler un comportement inapproprie :

  1. Accedez au profil ou a la conversation
  2. Appuyez sur le menu (trois points)
  3. Selectionnez "Signaler"
  4. Choisissez la raison du signalement
  5. Ajoutez des details si necessaire

Conseils de securite pour les rencontres

  • Rencontrez toujours dans des lieux publics
  • Informez un proche de vos projets
  • Faites confiance a votre instinct
  • Ne partagez pas d'informations personnelles sensibles
  • Evitez de monter dans le vehicule d'un inconnu

Parametres

  • Changer le mot de passe : Securisez votre compte

Notifications

  • Messages : Activez/desactivez les notifications de messages
  • Demandes de contact : Soyez alerte des nouvelles demandes
  • Flash Meetups : Notifications pour les meetups a proximite

Confidentialite

  • Visibilite du profil : Controlez qui peut voir votre profil
  • Localisation : Gerez le partage de votre position
  • Donnees : Telechargez ou supprimez vos donnees

Application

  • Langue : Choisissez la langue de l'interface
  • Theme : Mode clair ou sombre
